# mt32-pi-gameport-hat

Raspberry Pi hat for [mt32-pi](https://github.com/dwhinham/mt32-pi) with the following features:
* 128x32 OLED display
* PCM5102 DAC
* 4 buttons
* MIDI input via DA-15 gameport
* Joystick passthrough
